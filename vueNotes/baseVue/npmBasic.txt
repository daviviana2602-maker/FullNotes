==================== VUE 3 ====================


Vue é um Framework JavaScript para construção de interfaces (Frontend).


Responsabilidades:

- Renderizar dados na tela
- Capturar interação do usuário
- Consumir APIs
- Controlar navegação entre páginas
- Gerenciar estado da aplicação



==================== FLUXO DE CONSUMO ====================


Usuário abre a página

        ↓

Vue faz GET /products

        ↓

Spring retorna JSON

        ↓

Vue salva os dados em memória

        ↓

Template renderiza a lista

        ↓

Usuário visualiza os produtos
