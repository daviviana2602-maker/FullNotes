================ Funcionamento Interno ================

Service:
throw new NotFoundException("Bolo não encontrado");

        ↓

O construtor da Exception recebe:
message = "Bolo não encontrado"

        ↓

Executa:
super(message);

        ↓

O RuntimeException guarda essa mensagem internamente.

        ↓
        
Mais tarde no GlobalExceptionHandler:
ex.getMessage()

retorna:
Bolo não encontrado


================ Funcionamento Para se adequar ao Frontend com melhor forma ================


public class ForbiddenException extends RuntimeException {


    private final String errorCode;      // errorCode precisa estar no ErrorResponse e precisa ser declarado, enquanto message já existe na RuntimeException


    public ForbiddenException(String errorCode, String message) {
        super(message);
        this.errorCode = errorCode;
    }


    public String getErrorCode() {
        return errorCode;

    }

}
