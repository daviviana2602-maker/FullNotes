==================== ESTRUTURA VUE + TS ====================

src/

api/
├── api.ts
│
│ Configuração central do Axios.
│ - baseURL
│ - Request Interceptors
│ - Response Interceptors
│ - Refresh Token automático
│ - Tratamento global de autenticação


router/
├── router.ts
│
│ Configuração das rotas da aplicação
│ - URLs
│ - View correspondente
│ - Guards (autenticação, admin, etc)


services/
├── AuthService.ts
├── ProductService.ts
├── UserService.ts
│
│ Camada responsável por consumir a API
│ Cada service monta as requests HTTP
│ envia DTOs tipados quando necessário e retorna os dados do backend


views/
├── LoginView.vue
├── ProductsView.vue
├── AdminView.vue
│
│ Páginas da aplicação.
│ Contêm template, estilização, lógica da página e chamadas para os services.
